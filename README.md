# Controle de Despesas

Projeto em constante desenvolvimento para estudos e aprendizado sobre boas práticas no desenvolvimento de software, padrões de desenvolvimento, novas tecnologias, etc.
Para essa finalidade estou utilizando um projeto de controle de despesas simples que armazenará as despesas, exibirá relatórios e consultas.
